TPMS TOOL

1. 가져오기 버튼 클릭 -> sn+im(include).xlsx 파일 열기(가져오기) ReadSuccess창이 나오면 성공 -> 확인 클릭.
2. 생산번호 버튼 클릭후 화면에 제대로 표시되는지 확인.
3. COM포트 확인후 연결. 자동 테스트시 자동 연결 체크 박스에 체크.(초기화시 바로 연결됨.)
4. 개별 테스트시 좌측의 1 ~ 14 박스 클릭.
4-1. LED 정상작동 확인시 육안검사 후 정상 작동시 성공, 실패시 실패 버튼 확인.
4-2. 검사완료및 초기화는 가장 마지막에 클릭 해야하며, 세트 전원이 종료된 것 을 확인후 클릭 요망.
5. 자동 테스트시 자동 테스트 시작 클릭.
5-1. 7번 LED 정상작동 확인시 성공,실패는 수동이며 육안검사 후 확인 요망.
6. 모든 검사가 끝나면 세트 전원 종료 후 해당 키트 제거후 다른 키트로 교체.
7. 검사 완료 및 초기화 클릭시 자동 연결 체크 박스에 체크 돼 있다면 자동연결.(초기화시 바로 연결됨.)
8. 생산번호 혹은 IMEI번호 입력후 생산번호 버튼 / IMEI 버튼 클릭시 지정된 SN가 화면에 나타남.(Serial Write)
8-1. 다음 번호 자동 넘김 체크박스 체크시 생산번호 버튼을 눌렀을때 자동으로 번호가 1씩 상승.
9. 위 상황 반복.
10. 모든 다운로드 및 검사가 끝나면 내보내기 버튼 클릭 -> license(export).xlsx 파일 열기(내보내기) -> write Success 창이 나오면 성공 -> 확인 클릭.
11. 엑셀창에서 파일 사용 가능 읽기/쓰기 클릭.
12. "성공"시트, "실패"시트 확인후 이상없음을 확인한 후 다른 이름으로 저장.

* 툴 실행 테스트 반응이 없으면 ComClear 버튼 활용.
** 툴 실행 도중 먹통이되거나 이상이 있으면 내보내기 클릭후 다른 이름으로 저장(필히 다른이름으로 저장).
